Privacy Policy for Auto Pinned Tabs

Effective Date: May 18, 2025

Auto Pinned Tabs is a Chrome extension designed to automatically open and pin specific tabs when the browser starts. We are committed to protecting your privacy and ensuring a transparent and secure experience while using our extension.

Information We Collect

This extension does not collect any personally identifiable information, financial information, health information, or authentication information.

The only data stored is your selected preferences (such as tab URLs) which are saved locally using Chrome's storage API.

How We Use the Permissions

  • storage: Used to save your tab preferences locally so the extension can remember which tabs to open and pin on startup.
  • tabs: Required to create and pin new tabs programmatically when the browser starts.

Data Sharing

We do not share, sell, or transfer any user data to third parties. No data leaves your device.

Remote Code

This extension does not use or load any remote code. All code is bundled within the extension and reviewed before deployment.

Your Consent

By using the Auto Pinned Tabs extension, you consent to this privacy policy.

Changes to This Policy

We may update this privacy policy in the future. If we do, we will post the revised version here with an updated effective date.
